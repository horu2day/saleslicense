### EGViewport

1.  **API 설명:**
    제공된 문서에 따르면 `HmEG.EGViewport`는 `HmEG.BatchedMeshGeometryConfig` 클래스의 생성자에 사용되는 매개변수 타입으로 언급됩니다.
    `HmEG.BatchedMeshGeometryConfig`의 생성자 시그니처는 다음과 같습니다:
    `#ctor(view, model, materialIndex, configIndex)`
    여기서 `view` 매개변수가 `HmEG.EGViewport` 타입입니다.
    문서에는 `EGViewport` 자체의 목적, 속성 또는 메서드에 대한 추가적인 정보가 없습니다. 단지 `BatchedMeshGeometryConfig`를 구성할 때 필요한 요소 중 하나임을 알 수 있습니다.

2.  **C# 코드 예제:**
    제공된 문서 내용만으로는 `EGViewport` 클래스를 직접 인스턴스화하거나 그 기능을 활용하는 완전하고 실행 가능한 C# 코드 예제를 작성하기 어렵습니다. 문서에는 `EGViewport`가 `BatchedMeshGeometryConfig` 생성자의 매개변수로 사용된다는 정보만 있으며, `EGViewport`를 생성하는 방법이나 다른 속성 및 메서드에 대한 정보가 누락되어 있습니다.

    따라서 `EGViewport`의 인스턴스를 생성하는 방법을 알 수 없으므로, 현재로서는 불가능합니다.

3.  **출처:**

4.  **주의사항:**
    제공된 문서 내용만으로는 `HmEG.EGViewport`의 구체적인 기능, 생성 방법, 속성 또는 메서드에 대한 정보가 매우 제한적입니다. `HmEG.BatchedMeshGeometryConfig` 생성자의 매개변수 타입으로만 확인되므로, 실제 개발에서 사용하기 위해서는 추가적인 HmEG 라이브러리 문서나 API 레퍼런스를 참조하여 `EGViewport` 클래스에 대한 더 자세한 정보를 확인해야 합니다.
